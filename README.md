## Installing

.NET Cli

```sh
dotnet add package QingYi.Core
```

Package Manager

```TEXT
NuGet\Install-Package QingYi.Core
```

Package Reference

Need specific the verison, 5.1.1 is the latest release version.

```xml
<PackageReference Include="QingYi.Core" Version="5.1.1" />
```
